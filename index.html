Administración y Seguridad de Redes

Este curso está orientado a formar profesionales en la administración de redes, planificación y detección de fallas de las mismas, así como también prepararlos para poder brindar soluciones seguras y confiables utilizando la tecnología a nuestro alcance y a estar preparados a medida que esta evoluciona. Cada módulo (nivel) se asegura de cubrir un segmento específicoal mismo tiempo que nos permite prepararnos para los siguientes pasos.


Objetivos del Curso:


- Identificar amenazas de seguridad a las que se expone una red.
- Garantizar integridad, disponibilidad y privacidad de datos y recursos.
